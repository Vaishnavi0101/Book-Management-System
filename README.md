Book Management System

Project Description
This is a simple Book Management System built using React and Vite.  
It allows users to perform CRUD operations on books using a mock API.

Features
- Add new books
- View all books
- Update existing books
- Delete books
- Search books by title or author
- Filter books by genre
- API integration using MockAPI or JSON Server
- Responsive user interface

Tech Stack
- React (Vite)
- JavaScript
- Axios
- CSS / Bootstrap
- MockAPI / JSON Server

How to Run the Project

1. Clone the repository
 Clone the repository (only if needed for new system)
git clone https://github.com/Vaishnavi0101/Book-Management-System.git

2. Go to project folder
cd BookManage

3. Install dependencies
npm install

4. Run the project
npm run dev

Build for Production
npm run build
